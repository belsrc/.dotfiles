# .dotfiles

Usual dotfile repo that everyone seems to have.
I switch between Windows and Mac, so the script is aimed towards Mac and Window Subsystem.

## Install

```bash
cd
git clone https://github.com/belsrc/.dotfiles.git
cd .dotfiles
bash setup.sh
```
